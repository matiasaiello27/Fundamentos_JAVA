# ☕ Fundamentos de Programación en Java

Este repositorio contiene una colección de programas básicos desarrollados durante mis primeros pasos en el ciclo de Desarrollo de Aplicaciones Multiplataforma (DAM). 

El objetivo de esta colección es demostrar el dominio de las estructuras de control de flujo, bucles y lógica fundamental en Java.

## 🛠️ Tecnologías y Conceptos

- **Lenguaje:** Java
- **Conceptos Aplicados:**
  - Estructuras condicionales (`if`, `else if`, `switch`).
  - Bucles iterativos (`for`, `while`, `do-while`).
  - Captura y validación de datos por consola (`Scanner`).
  - Operadores lógicos y matemáticos (ej: operador módulo `%`).

## 📂 Contenido del Repositorio

El repositorio consta de 5 scripts independientes, cada uno resolviendo un problema lógico distinto:

1. **Programa 1 (Ordenador de números):** Toma tres números por consola y los ordena de forma ascendente o descendente mediante un algoritmo manual de intercambio de variables.
2. **Programa 2 (Cuenta regresiva):** Implementación de un bucle `while` básico para generar una cuenta regresiva del 10 al 1.
3. **Programa 3 (Generador de secuencias):** Solicita un número positivo (con validación de entrada) e imprime los siguientes 20 números usando un bucle `for`.
4. **Programa 4 (Calculadora de IMC):** Calcula el Índice de Masa Corporal. Incluye validación estricta de rangos (peso entre 30-300kg, altura entre 1.3-2m) y clasifica el resultado en categorías de salud.
5. **Programa 5 (Detector de múltiplos):** Recorre los números del 1 al 100, detecta, imprime y contabiliza cuáles son múltiplos de 2 o de 3.
